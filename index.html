Customer Churn Analysis
GitHub ↗
End-to-end project · Excel → MySQL → Python → Power BI

Analysed 7,032 customer records to uncover why a telecommunication company was losing 26.6% of customers annually — above the 18–25% industry average. Built a complete four-tool pipeline, identified contract type and tenure as the biggest churn drivers, engineered a custom risk scoring model in Python that flagged 31 high-risk customers, and delivered a Power BI dashboard with recommendations projected to recover over $450K in recurring revenue.
